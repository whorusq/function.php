### 1. 函数列表

- [x] sys_curl() 通用的 curl 封装
- [x] sys_random_pwd() 生成随机字符串
- [ ] sys_num2rmb() 数值转人民币
- [ ] sys_verify_idcard() 验证身份证号，获取身份证信息

- [ ] sys_out() 接口数据输出：json/xml
- [x] sys_export_csv() 写入 CSV
- [x] sys_encrypt() 字符串加密
- [x] sys_decrypt() 字符串解密
- [x] sys_uuid() 生成唯一 ID
